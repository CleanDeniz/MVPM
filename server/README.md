# Server (Render-ready, Supabase)
1) Create a Render Web Service:
   - Runtime: Node 18+
   - Build command: `npm install`
   - Start command: `npm start`
2) Environment variables:
   - SUPABASE_URL
   - SUPABASE_KEY  (service role in dev, rotate later)
   - BOT_TOKEN
   - ADMIN_SECRET
   - CLIENT_ORIGIN (Vercel client URL)

Apply `schema.sql` to your Supabase project (SQL Editor).
